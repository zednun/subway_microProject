Welcome to subway project!

Usage:
'Subway' can be used in metro stations. There are three types of tickets that users can buy.
1- Expirable (It has expiration date)
2- Chargeable (It has some specific amount of charge) 
3- Disposable (It can be used once)
Also, each user would be able to connect to his/her bank account and deposit, withdraw, check the balance and ...

How to run the code?
1- chmod +x ./run.sh
2- Run run.sh. It will download all the dependencies and then run the main source.
